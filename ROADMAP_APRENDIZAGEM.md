# 🎓 Roadmap de Aprendizado - Desenvolvimento Web

## 📅 **FASE 1: Fundamentos (1-2 meses)**

### 🌐 HTML & CSS Básico
```bash
# O que aprender:
- HTML5 semântico (header, nav, main, section, footer)
- CSS Flexbox e Grid
- Responsividade com media queries
- Cores, tipografia, espaçamento

# Projeto prático:
- Landing page simples
- Portfólio pessoal
- Clone de site famoso

# Recursos:
- MDN Web Docs
- CSS Tricks
- Flexbox Froggy
- Grid Garden
```

### ⚛️ JavaScript Básico
```bash
# O que aprender:
- Variáveis, funções, arrays, objetos
- DOM manipulation
- Eventos (click, scroll, resize)
- Fetch API para requisições
- ES6+ (arrow functions, destructuring)

# Projeto prático:
- To-do list interativo
- Calculadora
- Weather app com API

# Recursos:
- JavaScript.info
- FreeCodeCamp
- The Odin Project
```

## ⏰ **Plano de Estudos Sugerido**

### 📅 Semanal (10-15 horas):
```bash
Segunda: Teoria (2h) + Vídeos (1h)
Terça: Prática guiada (2h)
Quarta: Projeto pessoal (3h)
Quinta: Debug e refatoração (2h)
Sexta: Review e planejamento (1h)
FDS: Projeto livre ou descanso
```

### 🎯 Metas Mensais:
```bash
Mês 1: HTML/CSS + JS básico
Mês 2: React fundamentals
Mês 3: Tailwind + Vite

Mês 5: APIs + integrações

```

## 🏆 **Dicas de Sucesso**

### 💡 Consistência:
- **1 hora por dia** > 7 horas no domingo
- **Pequenos projetos** > projetos gigantes
- **Documentar o aprendizado** (notas, blog)

### 🐛 Debug Mindset:
- **Erros são normais**, parte do processo
- **Google é seu melhor amigo**
- **Stack Overflow**: aprenda a fazer boas perguntas

### 🚀 Projects First:
- **Teoria só com prática**
- **Construa, quebre, conserte**
- **Mostre seu trabalho** (GitHub, portfolio)
